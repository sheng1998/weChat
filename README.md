##### 进入文件夹 `weChat-server`

1. 安装依赖

   ```shell
   npm install
   ```

2. 运行后台服务器(运行在3001端口)

   ```shell
   npm start
   ```

##### 进入文件夹`weChat-page`

1. 安装依赖

   ```shell
   npm install
   ```

   

2. 运行项目（默认端口号为8080）

   ```shell
   npm start
   ```

3. 打开页面

   ```
   http://localhost:8080/
   ```
